# 📦 SQL Server — Estudos e Prática

Repositório de estudos práticos em SQL Server, organizado por trilhas de aprendizado.
Desenvolvido durante jornada de preparação para as certificações **DP-300** e **DP-900**.

## 🧠 Metodologia pré-query

Antes de escrever qualquer consulta, responda:
1. **O que preciso ver?** — qual é o resultado esperado?
2. **Quais tabelas têm essa informação?** — localizar as fontes de dados
3. **O que essas tabelas têm em comum?** — identificar as chaves de JOIN
